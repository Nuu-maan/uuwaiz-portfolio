## Installation

1. Clone this repository:

git clone https://github.com/Nuu-maan/uuwaiz-portfolio.git


2. Navigate to the project folder:

cd uuwaiz.portfolio

3. Navigate to js folder
cd js

4. Start server
  node server.js


5. Open `index.html` in your browser.
